# Phase D · 心智模型提煉 + SKILL 組裝

> **階段目標**：把 `01-07.md` 的結構化調研轉成 1 個可運行的 `SKILL.md`，讓 AI 能「像股癌一樣思考」。
> **前置**：Phase C 完成，7 個 canonical 檔案都在 `references/research/`。

## D.1 心智模型提取（3-7 個）

讀 `references/extraction-framework.md` 裡的「三重驗證」方法論：
- **跨域複現**：出現在 ≥2 個不同領域 / 話題
- **生成力**：能推斷他對新問題的立場
- **排他性**：不是所有人都這樣想

### 程序

1. 從 `01-writings.md` 提出 15-30 個**候選論點**（有 coverage ≥ 3 批或 5 集以上的）
2. 對每個候選執行三重驗證：
   - 三重通過 → 心智模型
   - 僅 1-2 重 → 降級為決策啟發式
   - 0 重 → 丟棄
3. 按排他性強度排序，取 top 3-7
4. 每個模型記錄：
   ```
   ### 模型 N: [名稱]
   - **一句話描述**: [什麼是這個模型]
   - **跨域證據**: 
     - 應用於 A 領域（EPxxx 原句）
     - 應用於 B 領域（EPxxx 原句）
   - **生成力範例**: 對 [新問題 X] 可推斷他會怎麼判斷
   - **局限**: [在哪些情況失效 / 他自己承認過的盲點]
   ```

### 候選心智模型（Phase A 已偵測到的蛛絲馬跡）

- **多元配置論 / 工具箱思維**（EP102 / EP180 / EP100）
- **柳樹理論**（EP57 首提、需追蹤後續複現）
- **灰階思考 / 反二元腦**（他的書名、EP100 批評線性思考）
- **損害控制優先於看對**（EP169）
- **逆選情緒波動 / 阿呆谷**（EP99 / EP100）
- **快速致富必有風險**（EP327 / 貫穿全期反詐）

Map agent Phase B 會補充更多候選，Phase D 主 agent 根據資料驗證 / 淘汰。

## D.2 決策啟發式提取（5-10 條）

格式「如果 X，則 Y」，需要具體 EP 案例支撐。例如：
- 「如果某支股票公司基本面爛但大家狂 hype → 提醒聽眾危險」
- 「如果我看不懂的金融工具 → 不碰並公開批評」
- 「如果 8/5 黑天鵝 → 先看是否流動性危機、再決定」

從 `05-decisions.md` 歸納。

## D.3 表達 DNA 分析

對照 nuwa `skill-template.md` 的 6 維度從 `03-expression-dna.md` 抓：

| 維度 | 股癌專案預期發現 |
|---|---|
| 句式偏好 | 預期：長句、常省略標點、一氣呵成、問句 + 自答 |
| 詞彙特徵 | 預期高頻：主委 / 幹你娘 / 他媽的 / 應該 / 就是 / 沒有 / 可能 / 老實講 / 其實 |
| 節奏感 | 預期：先鋪陳再結論 / 常轉折 / 用「那」「所以」過渡 |
| 幽默方式 | 預期：自嘲 + 挖苦 + 荒誕對比（例：「蝦子在摸象」）+ 台灣口語梗（秋口幹你娘） |
| 確定性表達 | 預期：混合 — 金融技術面自信、對市場方向常帶「我不一定對」|
| 引用習慣 | 預期：數據、財報、國際新聞、自己的朋友 / 觀眾故事、電影（魔戒 / 哈比人）|

## D.4 價值觀與反模式

- **價值觀 3-5 條**：從 `07-life-events.md` 抽（他的家庭 / 事業 / 送別小徐 的行動揭露）
- **反模式**：他明確反對的（功德 / 財富自由話術 / 快速致富 / 基金廣告念超快 / 逢中必反的二元腦）
- **內在張力**：從 Phase C 的衝突報告搬過來

## D.5 智識譜系

從訪談 + 引用習慣推：
- 受誰影響（費曼 / Buffett / 巴菲特 / Hastings / Munger / Damodaran？）
- 影響了誰（台灣投資播客次文化、主委文化）
- 位置（台灣金融口語化內容創作者的原點）

## D.6 誠實邊界（nuwa 強制）

必寫：
- 不能預測他對 2026-04 之後新問題的反應
- 不能替代股癌本人創造力 / 直覺
- 公開 podcast 表達 vs 真實想法可能有差距（他多次自嘲過）
- 調研截止時間：**2026-04-23**（Phase A 起跑日）
- **外部批評視角缺失**：本專案純本地語料，對於「別人怎麼批評他」非常薄弱
- 生活事件可能有錯誤日期 / 錯誤歸因（蒸餾團隊非本人）

## D.7 SKILL.md 組裝

### 步驟

1. **讀模板**：`references/skill-template.md`（nuwa 原始模板）
2. **目錄位置**：最終 SKILL.md 寫入 `~/.claude/skills/gooaye-perspective/SKILL.md`（**根目錄，不在 references/**）
3. **填充 section**（對照 nuwa SKILL.md Phase 3 Step 2 表格）
4. **特別生成「回答工作流 Agentic Protocol」**：

```markdown
## 回答工作流（Agentic Protocol）

**核心原則：股癌不凭感覺說話，他會先查再講**

### Step 1: 問題分類
| 類型 | 特徵 | 行動 |
|---|---|---|
| 需要事實 | 涉及某公司 / 股票 / 數據 / 人物 | → Step 2 |
| 純框架 | 人生 / 哲學 / 方法論 | → Step 3 |
| 混合 | 用具體案例談抽象 | → 先事實再框架 |

### Step 2: 股癌式研究（**這段每次都要重新推導，對齊心智模型**）
- **看基本面**：營收 / 財報 / 產業週期
- **看市場情緒**：VIX / AAII / 散戶行為
- **看歷史類比**：這情況像哪次 cycle
- **逆向問**：如果大家都錯會怎樣 / 如果他自己錯會怎樣
- **看配置角度**：這是 all-in 還是只是 20% 配置？

### Step 3: 股癌式回答
- 用他的 DNA 格式寫 —— 長句 + 轉折 + 自嘲
- 情緒適當時加髒話
- 最後必帶「我不一定對」「自己評估風險」
```
