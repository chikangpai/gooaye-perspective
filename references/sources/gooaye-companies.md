# 謝孟恭的兩家投資公司

> ⚠️ **關鍵修正**：不是只有一家「肖楠資本」。謝孟恭成立了**兩家**投資公司：**肖楠資本** 和 **烏杉資本**，命名都取自「台灣針葉五木」（肖楠、烏杉均為其中一員）。兩家為**早期 → 中期接棒**的雙層架構。

## 對照表（一眼看懂）

| | 肖楠資本 | 烏杉資本 |
|---|---|---|
| 英文名 | **Calocedrus Capital** | **Lunta Ventures** |
| 公開時間 | **2025/8/12** | **2025/11/5** 於 TCCF 創意內容大會 MOU |
| 正式登記 | 2025/4/29 | 待確認 |
| 投資階段 | 已有 Demo、開發中（**中期陪跑**） | **種子輪 First Check**（Idea → Demo） |
| 基金型態 | **Evergreen Fund**（常青基金、無固定退場年限） | 與文策院共營**數位遊戲基金** |
| 關鍵合作 | 嘖嘖 + 芒果派對 + GameWorks | **文策院 TAICCA（國家隊）** + 嘖嘖 + 好翊投資 |
| 規模目標 | AUM 3 億（2026 年底） | 未公開 |
| 角色定位 | 中期陪跑資本 | **First Check** 第一棒 |
| 命名 | 台灣針葉五木「肖楠」— 育才如種樹、長期耐心 | 台灣針葉五木「烏杉」— 深根厚植、持續成長 |

**簡記**：**烏杉**=更早（種子輪，國家隊在一起）、**肖楠**=稍後一點（Demo 之後，常青基金、民間）。

## 肖楠資本 · Calocedrus Capital

### 基本資訊
- **資本總額**：1.5 億新台幣
- **實收**：4,550.1 萬
- **董事長**：謝孟恭（持股 2,300,100 股、最大出資人）
- **董事**：
  - 徐震（嘖嘖共同創辦人，代表法人嘖星）
  - 楊奇紘（代表法人芒果派對，遊戲發行）
- **監察人**：李易鴻（代表法人曼潔拉）

### 投資定位
- 已經有團隊、有 Demo、在開發階段的**原創遊戲**
- **Evergreen（常青）基金**架構，沒有固定退場年限，可陪跑
- **2026 年底 AUM 目標新台幣 3 億**
- **首波已簽 4 筆**：三家遊戲公司 + 一款原創遊戲
  - 杯狗遊戲《紅眼露比》
  - 本氣遊戲《Night Rampage》
- 與 GameWorks（台灣遊戲孵化/合製單位）長期合作、提供開發諮詢、發行媒合

## 烏杉資本 · Lunta Ventures

### 基本資訊
- **董事長**：謝孟恭
- **董事**：
  - 徐震（嘖嘖共同創辦人）
  - 張聖翊（好翊投資負責人）
- **合作對象**：**文化內容策進院（文策院 TAICCA）**——即「國家隊」

### 投資定位
- **最早期 / 種子輪（Seed Stage）**：鎖定還在「Idea → Demo」、還在燒錢開發的團隊
- **定位為新創團隊的 "First Check"**（第一位機構投資人）
- 明確是想補上「好團隊死在沙灘上」、「被迫把 IP 賣斷給國外」的**資金斷層**
- **主軸**：與文策院共同啟動「數位遊戲基金」，協助原創 IP 走向國際

## 為什麼這對蒸餾 skill 很重要

這兩家公司揭示了 2025 年謝孟恭的重大人生 pivot：從「純粹的內容創作者 / 投資評論員」→「**做事的人**」。這種身份轉換會在 B11-B12 的談話中留下深刻指紋：
- 對「台灣創作 / IP / 遊戲產業」的態度變化
- 對「創業 / 風險承擔」的語言傾斜
- 對「資本配置 / fund management」的新觀點
- 與「文策院 / 嘖嘖 / 好翊 / GameWorks」等機構 / 人的引用增加
