## 用戶建議的「股癌新手補課順序」

Map agent 在 reduce 時把這個順序寫入最終 SKILL 的「**入門 primer**」段落：

1. **EP20 財你個鳥富自由** — 話術免疫力
2. **EP102 小朋友才做選擇** — 資產配置、不全押單一信仰
3. **EP169 損害控制** — 風控與停損
4. **EP180 走火入魔** — 不迷信單一方法
5. **EP327 快速致富必有風險** — 補一刀：太好聽的最危險
6. **EP260 / EP275** — 槓桿 ETF / 期貨（進階）
7. **EP117 / EP339 / EP550** — 梗曲（補社群文化）

## 三大自創術語候選 ⚡

以下三個詞可能是**獨創術語**（符合 nuwa 三重驗證的「自創概念」），Map agent 需在 extract 時**跨集追蹤反覆出現**以確認：

1. **柳樹理論**（EP57 首次出現）— 形容某類市場現象
2. **工具箱思維 / 走火入魔**（EP180）— 反對單一分析方法的方法論
3. **阿呆谷**（EP100 已出現）— 形容恐慌砍在最低點

## Map agent 指示

對上述 11 個經典集數，除了標準 5 檔輸出（`claims.md`、`expressions.md`、`decisions.md`、`life-events.md`、`timeline.md`）外：

- **額外生成 `classics/EPxxx-deep.md`**：每集寫 300-500 字的深度摘要，包含：
  - 本集核心論點 1-3 條（帶原句）
  - 使用的術語 / 類比 / 比喻
  - 股癌在該集的情緒狀態（冷靜？爆氣？自嘲？）
  - 若是梗集：梗的完整脈絡

這些深度摘要在 Phase D 組裝 SKILL 時會被用作「示例 dialogue」的來源。
